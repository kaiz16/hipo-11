 * atoms (smallest component)
    - input 
    - button
    - links 
    - menu-items
    - logo

 * molecules (combine atoms = one molecule)
    - form 
    - menu 
    - card

 * organisms (combine molecules = one organism)
    - navbar
    - footer
    - sidebar

 * templates (layout = components combined together)
    - skeleton

 * pages (final page)
    - pages like home, profile
